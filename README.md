# 🤖 AI Agent with RAG

A simple AI Agent built using **Python**, **Gradio**, and **RAG (Retrieval-Augmented Generation)**. It retrieves relevant information from uploaded documents and answers user queries based on the document content.

## 🚀 Features
- 📄 Upload PDF documents
- 🔍 Semantic search with embeddings
- 💬 Interactive chat interface
- ⚡ Fast and accurate document retrieval
- 🧠 Context-aware question answering

## 🛠️ Tech Stack
- Python
- Gradio
- LangChain
- FAISS
- Sentence Transformers
- PyPDF
